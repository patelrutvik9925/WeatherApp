# Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript. The application fetches real-time weather data using the Open-Meteo API and displays weather information for any city entered by the user.

## Features

* Search weather by city name
* Real-time weather data
* Displays temperature, humidity, and wind speed
* Weather condition descriptions with icons
* Input validation
* Error handling with notifications
* Responsive design
* Clean and modern user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Open-Meteo Weather API
* Open-Meteo Geocoding API

## How It Works

1. Enter a city name.
2. The Geocoding API converts the city name into latitude and longitude coordinates.
3. The Weather API fetches weather information using those coordinates.
4. The weather details are displayed on the page.

## Weather Information Displayed

* City Name
* Weather Description
* Temperature (°C)
* Humidity (%)
* Wind Speed (km/h)
